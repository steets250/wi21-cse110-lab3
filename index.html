HTML

Main root

<html>

The html tag represents the top level element of an HTML document. It encapsulates all of the HTML text in a file.

Document metadata

<head>

The head tag contains metadata about the html page, including the following three elements.

<title>

The title tag sets the text that is displayed in the browser tab or window.

<meta>

The mega tag contains site metadata that can't be expressed using the other tags.

<link>

The link tag is used to reference external files, including css stylesheets or site icons.

Content sectioning

<body>

The body tag contains the actual content of the html page, and can include all of the remaining elements.

<main>

The main tag indicates that the majority of the page content will be housed in this section.

<h1>

The h1 tag is the first highest level of section headings. It is commonly used for page/site titles.

<h2>

The h2 tag is the second highest level of section headings. It is commonly used for main headings.

<h3>

The h3 tag is the third highest level of section headings. It is commonly used for sub headings.

<section>

The section tag represents a seperate section of content, and can be used to denote different parts of a page.

Here is an example of a sentence that is housed in a seperate section than the other content.

<header>

The header tag contains introductory or beginning content of a page, like a site introduction or navigation content.

<footer>

The footer tag contains concluding or ending content of a page, like information about the author or copyright information.

<aside>

The aside tag is used to seperate a chunk of content that is indirectly related to the main content.

Here is an example of a sentence that is marked as being an aside, since it doesn't relate to the surrounding content.

<nav>

The nag tag denotes a section of the site that provides website navigation.

Here is an example of an area where I could link to other pages that this site has.

Text content

<div>

The div tag is used to denote a section of content. It has no effect on the layout, and is usually styled with CSS.

<span>

The span tag is similar to a div, but it represents an inline denotion of a section.

Here is an example of content inside a span, which can be styled differently than surrounding text.

<hr>

The hr tag is used to create a break in content. It stands for horizontal rule.

<p>

The p tag represents a paragraph, and are usually used to seperate paragraphs of text from each other.

<ul>

The ul tag is used to create an unordered list of items. It contains <li> items.

<figure>

The figure tag represents a piece of self contained content, which commonly has a caption attached.

<figcaption>

Images and multimedia

<img>

The img tag is used to embed images in a website.

<picture>

The picture tag can contain multiple images, and display different images based on preset conditions.

<iframe>

The iframe tag can embed a webpage within another webpage.

<audio>

The audio tag can be used to embed audio on a webpage.

<video>

The video tag can be used to embed video on a webpage.

<source>

The source tag is used to provide sources for a picture, audio, or video element It can be seen in use in the above example.

<canvas>

The canvas tag is used to draw graphics on a webpage.

Here is an example of text displayed within a canvas.

<svg>

The svg tag can be used to display vector graphics on a webpage.

Forms

<form>

The form tag contains a group of interactice elements that can be used to input user data.

Fieldset Legend

<optgroup>

The optgroup tag is used to contain a group of options for the select element.

<label>

The label tag is used to create a caption for an input.

<fieldset>

The fieldset tag is used to group multiple control types together.

<legend>

The legend tag is used to create a caption for a group of inputs.

<input>

The input tag is used to create interactive form controls.

<textarea>

The textarea tag is used to input a multiline text value.

<datalist>

The datalist tag provides a set list of allowable inputs for an option tag.

<select>

The select tag is used to receive input from a list of options and option groups.

<option>

The option tag is used to define an option for a select or optgroup tag.

Table

<table>

The table tag is used to contain a table of data.

<tr>

The tr tag represents a table row.

<td>

The td tag represents table data.

<th>

The th tag represents a table header cell.

<caption>

The caption tag representents the caption for a table.

<col>

The col tag is used to define a column type.

<colgroup>

The colgroup tag is used to define a group of columns in a table.

Interactive elements

<details>

The details tag creates hidable content which can be accessed with a dropdown.

Detail Summary Here is an example of text that has been hidden with a details tag.

<summary>

The summary tag specifies the text that will be used to indicate the contents of a details tag, and can be seen in the above example.

<button>

The button tag is used to create a clickable button, and can be used in forms or anywhere else where a click action is needed.

<template>

The template tag is used to contain HTML elements that are displayed as a result of a Javascript call, and aren't immediately visible.
